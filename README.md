# ddi
System to find drug interactions

As data the following DDI corpus was used:
Maria Herrero-Zazo, Isabel Segura-Bedmar, Paloma Martinez, Thierry Declerck, The DDI corpus: An annotated corpus with pharmacological substances and drug–drug interactions, Journal of Biomedical Informatics, Volume 46, Issue 5, October 2013, Pages 914-920, http://dx.doi.org/10.1016/j.jbi.2013.07.011. 

It can't be obtained [here](http://labda.inf.uc3m.es/doku.php?id=en:labda_downloadddicorpus)
